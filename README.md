# SAE14 – Création d'un site web

## Description du projet
Ce projet a été réalisé dans le cadre de la SAÉ14. Il s'agit d'un site web multipages en HTML et CSS, respectant les standards du web et les critères d'accessibilité.  

Le thème choisi est les aurores boréales, permettant de présenter des informations sur ce sujet et de mettre en valeur du contenu visuel et textuel.

## Objectifs
- Créer un site web adaptatif (Responsive Design) affichable sur ordinateur, tablette et mobile.
- Utiliser Flexbox et/ou CSS Grid pour la mise en page.
- Respecter la norme WCAG 2.0 AA pour l'accessibilité.
- Appliquer une charte graphique cohérente sur toutes les pages.
- Ajouter au moins une animation CSS.
- Déposer le code source sur GitHub et héberger le site sur GitHub Pages.

## Structure du site
Le site comportera 3 pages :  

1. Une page d'accueil
2. Une page de contenue
3. Une page à propos

## Technologies utilisées
- HTML5
- CSS3 (Flexbox, Grid)
- Git / GitHub
